要求：
1. 实现加减乘除四则运算。
2. 建立四个工程ABCD，A为lib，完成加法运算；B为dll，完成减法运算；C为dll，完成乘法运算；D为exe，完成除法运算。
D中存在加减乘除的接口，其中加法接口以调用A实现；减法接口采用在工程中配置加载B的方式实现；乘法接口以运行时加载C的方式实现；除法接口调用本身函数的方式实现。
3. 使用git做代码管理，四则运算分别开4个分支，最后合并分支。


四则运算思路：
	在master下创建四个分支，进行编程新建不同文件，实现各自功能。
	addition:        lib 
	subtraction:   dll (静态加载)
	multiplication:       dll (动态加载)
	division          :exe (empty_project)       调用 以上 完成加减乘运算
	最后合并分支。
